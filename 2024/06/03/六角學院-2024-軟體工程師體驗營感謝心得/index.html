六角學院 - 2024 軟體工程師體驗營感謝心得

參加體驗營動機 🤔

之前有學過前端開發，不過學的很不是很專精且很痛苦，因為當時覺得看不懂專有術語常常有撞牆期，而且因為一段時間沒接觸寫 code 後就變得很害怕再接觸。

某陣子在臉書一直不斷被可愛又帥氣的洧杰校長拍了多部搞笑的視頻洗眼睛 ( ? ) 吸引了我的目光，一個月的體驗營只要 799 元就可以學到前端三寶、包含 jquery、Bootstap、Hexo 部落格網頁開發以及切版操作，更棒的是有助教和洧杰校長的協助，直播課程的互動和小組同學互動交流，能不香嗎？

抱持著俗又大碗又沒有什麼損失的想法，決定再給自己一次機會衝一波。


課程學習內容 🧠

  • 第一週 - 打造最強前端開發環境

    學習 HTML5 架構，用奇摩新聞練手，再搭配 CSS3 來調整介面樣式，並利用flex 卡片排版技巧來完成證書任務作業一：一個人品牌網站 - 開發經驗卡片設計。

  • 第二週 - FLEXBOX 多欄式排版全攻略
    加強　FLEXBOX 的概念，練習如何將　FLEXBOX 運用的更完善，並且利用 flex 修練時光機 10 個練習讓自己對此可以瞭解更根深地步。

    Felx 修練時光屋
    Flex 修練時光屋任務挑戰

  • 第三週 - 響應式網頁設計
    學習 RWD 斷點的運用，排版的概念，切版的注意要點以及禁忌，以及權重層級的規則和它的重要性。將第二週與第三週所學的內容運用在切版任務作業二 - 個人品牌網站，這時就到了要考驗你對初期 HTML、CSS 所學的功力如何了!

  • 第四週：Hexo 極速免費架站全攻略
    　　利用 Hexo 套件和 Git 指令一起架設出靜態網頁，並放上 GitHub 免費線上程式碼雲端託管平台，做一個專屬於自己的免費部落格，免錢的～超香！ 👍🏻

Hexo 部落格架站成功在這裡


學習經歷 👩🏻‍🔧

這次的經驗竟然出乎我的意料，最大的驚喜就是我竟然不害怕撞牆期了！我在大家的身上學到遇到問題發生時的從容不迫的態度，讓我遇事不再害怕出錯，出錯了學習改進後才會進步，冷靜地找到問題的解答。這次參加體驗營，我很感謝幫助我的洧杰校長、助教們以及同學們，只要我有疑問在社群上提問，馬上就有人可以回答我，甚至線上連線畫面直接幫我解答我的疑難雜症。

我也發現我對於前端開發內容學習來不在是處於一直在不懂半懂的狀態，我學會將我的問題提問方式可以更直觀切入重點，就算別人看不懂我的問題思維是什麼，只要我在釐清自己的問題方式，其他人都還蠻有耐性的為我解惑。

我是屬於偏需感管型情緒價值必須拉滿的學習方式，那種傳統制式化的「你為什麼不會」、「老鳥看菜雞噹」的方式在體驗營學習過程中完全都沒有發生這種情況，反倒是是一種讓我很舒心的學習環境。而且意外的發現同學們都好強好專業喔！在作業問題解答區內回答問題都馬上回答到點上，力馬快速 closed 每一單個疑問，不得不由衷的佩服。洧杰校長也是很親切很接地氣的搞笑，每次聽到他在直播間喊「ㄟ～同學！」就很開心啊！而且六角學院的學習終旨就是『只要你不放棄自己，六角學院就不會放棄你』，真的讓人深深地覺得不可以放棄學習不能放棄轉職的初心是什麼！


總結心得 🥊

再次，很感謝洧杰校長、助教們和同學們，讓我在這一個月的體驗營中過得很愉快充實，大家都給我很溫暖貼心的感覺。讓我不在有害怕而停滯不敢往前邁進一步待在十字路口迷茫的感覺。接下來我會參加另一個介面設計的訓練營，待完成後我會再繼續向 React 學習，期待我轉職成功到來的一天。

大家一起加油！❤️‍🔥❤️‍🔥❤️‍🔥
